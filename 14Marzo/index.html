Los elementos de HTML se catalogan como elementos "en bloque" o elementos "en linea"

Elementos de bloque

Los elementos en bloque ocupan todo el espacio que tengan disponible, generando un "bloque"
Pueden contener elementos de linea y otros elementos de bloque

En linea

Ocupan solamente el espacio requerido para desplegar su contenido
Pueden contener unicamente datos y otros de linea

Es posible insertar contenido en un elemento de bloque con ayuda de un elemento de linea
